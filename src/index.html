How the game works

In Spot It (Dobble), any two cards share exactly one matching symbol. This is based on finite projective planes. A deck can be generated with n symbols per card, producing n² - n + 1 cards.

  • Each card has exactly 8 symbols in the classic game.
  • Any pair of cards has one and only one shared symbol.
  • Your custom deck can be smaller for easier printing.
